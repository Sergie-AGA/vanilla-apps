Custom Sticky Notes

Custom Sticky Notes is a small project that allows you to create notes that possess an image, a title and a description. Currently unavailable for mobile use. Upcoming features include font size and note proportions.

Simply click the "new sticky note" button to start creating. To make the image work, you need the address that has the extension (.jpg / .png / etc) in it.

Custom Sticky Notes is part of "Mini-Apps by Sergie", an ongoing project aimed at showing how to make small web applications
